# Exam Instructions for Students

## Special Question Types

### 1. Mathematical Questions with Calculator
- Questions marked with **"Calculator Allowed"** badge permit calculator use
- You can use your system calculator (Windows Calculator, macOS Calculator, etc.)
- Calculator usage will NOT trigger security alerts for these questions
- Close calculator when done to avoid accidental alerts on other questions

### 2. Graph/Plotting Questions
- Draw graphs on paper
- Take a clear photo of your graph
- Upload the image in the answer area (if file upload is enabled)
- Alternatively, describe your graph in detail in the text area

### 3. Programming/Code Questions
- Code questions have a dark code editor with syntax highlighting
- Write your code directly in the browser
- Your code is auto-saved as you type
- Supported languages: Python, Java, C++, JavaScript, C
- Code will be manually graded by your invigilator
- You can use local IDE and paste final code (copy-paste is allowed for code questions)

## General Exam Rules
- Stay in fullscreen mode throughout the exam
- Keep your face visible to the camera
- Do not switch tabs or windows (except for whitelisted calculator)
- Do not use external resources unless explicitly allowed
- Timer will auto-submit when time expires

## Technical Support
- Click "Help" button during exam for immediate assistance
- Contact your invigilator before exam if you have technical issues
